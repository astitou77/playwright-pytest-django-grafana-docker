# playwright-pytest-django-grafana-docker
playwright test framework using pytest with a django front-end, prometheus/grafana reporting, docker containerization, and a CI/CD pipeline

# HOW TO RUN TESTS : 

> pytest tests/eTA

> pytest tests/eTA/test_eTA_smoke_test.py::test_fill_ETA_Page

# DEBUG

> pytest --pdb

> pytest --trace

> pytest -s
